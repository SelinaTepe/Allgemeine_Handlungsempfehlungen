Sichere Kommunikation

Egal ob ob Sie Telefonieren, Viedeoanrufe tätigen oder Textnachrichten über Messenger versenden, die Kommuniktion über das Internet ist größtenteils unverschlüsselt und somit ein beliebtes Angriffsziel, um persönliche Nachrichten abzugreifen. Die nachfolgenden allgemeinen Handlungsempfehlungen beiten eine gute Basisabsicherung, um sicher im Internet zu Kommunizieren.

  1. Achten Sie auf eine sichere Verbindung: Stellen Sie sicher, dass bei der Nutzung von E-Mail-Programmen im Browser stets eine durchgängige HTTPS-Verbindung besteht. Überprüfen Sie in der Browserleiste das Schlosssymbol und das „https“
  2. Nutzen Sie Messenger-Dienste mit Ende-zu-Ende-Verschlüsselung: Wählen Sie nur Dienste, die eine Ende-zu-Ende-Verschlüsselung unterstützen, um die Sicherheit Ihrer Nachrichten zu gewährleisten.
  3. Verwenden Sie bewährte E-Mail-Clients: Nutzen Sie für das Senden und Empfangen von E-Mails bekannte und weit verbreitete E-Mail-Programme.
  4. Schützen Sie sensible Inhalte: Versenden Sie E-Mails mit vertraulichen Informationen ausschließlich verschlüsselt, beispielsweise mit PGP oder S/MIME.
  5. Verwenden Sie etablierte Messenger: Kommunizieren Sie über Messenger-Dienste, die weithin etabliert sind und Sicherheitsstandards einhalten.
  6. Aktivieren Sie den Spamfilter: Schalten Sie in Ihrem E-Mail-Programm den Spamfilter ein, um unerwünschte und potenziell gefährliche E-Mails auszusortieren.
  7. Setzen Sie auf sichere Protokolle: Stellen Sie sicher, dass Ihr E-Mail-Programm nur sichere Übertragungsprotokolle wie POP3S, IMAPS und SMTPS verwendet.
  8. Deaktivieren Sie externe Inhalte: Blockieren Sie das automatische Laden externer Inhalte wie Bilder in HTML-E-Mails, da sie schädlichen Code enthalten könnten.
  9. Prüfen Sie den Absender: Überprüfen Sie E-Mails auf die tatsächliche Absenderadresse, indem Sie den Mauszeiger über die E-Mail-Adresse bewegen. Öffnen Sie keine Nachrichten von unbekannten oder verdächtigen Absendern.
  10. Seien Sie vorsichtig mit Anhängen: Laden Sie keine E-Mail-Anhänge herunter, die ungewöhnliche Dateiendungen wie .exe oder doppelte Endungen wie .pdf.exe haben.
  11. Klicken Sie keine Links an: Vermeiden Sie es, Links in E-Mails anzuklicken. Diese könnten zu gefälschten Webseiten führen, die darauf abzielen, Ihre Zugangsdaten zu stehlen.
  12. Löschen Sie verdächtige E-Mails: Öffnen Sie keine E-Mails, deren Legitimität Sie anzweifeln. Löschen Sie diese umgehend.
  13. Achten Sie auf schlechte Rechtschreibung: Gefälschte E-Mails sind oft an schlechter Grammatik und Rechtschreibung zu erkennen.
  14. Laden Sie keine unbekannten Dateien herunter:Vermeiden Sie es, Dateien aus unbekannten Quellen zu öffnen oder herunterzuladen.
  15. Blockieren Sie verdächtige Kontakte: Blockieren Sie Kontakte, wenn Sie Zweifel an ihrer Identität haben.
  16. Deaktivieren Sie das automatische Herunterladen: Schalten Sie in Ihrem Messenger-Dienst die automatische Download-Funktion für Bilder und Dateien aus, um das Risiko schädlicher Inhalte zu minimieren.
